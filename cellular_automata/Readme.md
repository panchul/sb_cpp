# cellular automata

To build:

    $ g++ cellauto.cpp

Simple example of basic pattern:

    k 4; r1; n 10; mapping: 3311100320
    0->0
    1->2
    2->3
    3->0
    4->0
    5->1
    6->1
    7->1
    8->3
    9->3
